A Set-Theoretic Approach to Establishing Bijections between Integers and Rational Numbers

Anonymous Authors
March 31, 2025
Abstract: This paper explores a novel set-theoretic approach to establishing a bijection between integers and rational numbers. We introduce two key concepts: a measure on collections of sets (μ) that maps to rational numbers, and a binary encoding rule (χᵢ) that maps to integers. We analyze the potential for creating a bijection through this dual mapping and discuss challenges and possible solutions. We also explore the relationship between collections of sets and binary sequences, establishing a bijection between certain collections of sets and binary sequences, then demonstrating how to recover the original set collection from a given binary sequence.

1. Introduction

The countability of rational numbers is a well-established result in set theory. Standard proofs utilize techniques such as Cantor's diagonal argument, the "snake" enumeration, pairing functions, or applications of the Schröder-Bernstein theorem. This paper proposes an alternative approach using set- theoretic constructions and measure theory to establish a bijection between integers (represented as binary strings) and rational numbers. We also explore how binary sequences can be derived from collections of sets and vice versa, providing a foundation for our bijection approach.

2. Definitions and Notation

2.1 Set Collections

Let Å = {A₁, A₂, ..., Aₙ} be a finite collection of finite sets, where each Aᵢ ⊂ ℕ.

2.2 Set Union Ratio

Definition 2.1 (Set Union Ratio).

The Set Union Ratio μ: 𝒫(𝒫(ℕ)) → [0,1] is defined as:

μ(Å) = |∪A∈Å A| / ∑A∈Å |A|

This measure represents the ratio between the cardinality of the union of all sets in the collection and the sum of individual set cardinalities.

Example 2.1 (Set Union Ratio).

Consider a collection Å = {{1}, {1,2}, {1,2,3}}:

|∪A∈Å A| = |{1,2,3}| = 3

∑A∈Å |A| = 1 + 2 + 3 = 6

Therefore, μ(Å) = 3/6 = 0.5

2.3 Binary Encoding Rule

Definition 2.2 (Binary Encoding Rule).

The Binary Encoding Rule χᵢ: 𝒫(𝒫(ℕ)) → {0,1} is defined for each position i ∈ ℕ as:

χᵢ(Å) = 1 if i = n + |Aₙ| - 1 for some Aₙ ∈ Å
χᵢ(Å) = 0 otherwise

This generates a binary representation of the collection Å.

Equivalently, for a collection of sets A = {A₀, A₁, ..., Aₘ₋₁}, we define a binary sequence χ = (χ₀, χ₁, ..., χₙ₋₁) where:

  • χᵢ = 1 if i is one less than a cumulative sum of set cardinalities, i.e.,
    i ∈ {|A₀|-1, |A₀|+|A₁|-1, |A₀|+|A₁|+|A₂|-1, ..., Σⱼ₌₀ᵏ |Aⱼ|-1 for k=0,1,...,m-1}
  • χᵢ = 0 otherwise

This creates a mapping between collections of sets and binary sequences where the positions of 1s in the sequence correspond to the cumulative sums of set cardinalities (minus 1).

Example 2.2 (Binary Encoding).

Consider a collection Å = {{1}, {1,2}, {1,2,3}}:

For each set Aₙ ∈ Å:

Set A₀: n=0, |A₀|=1, so i=0+1-1=0 → χ₀=1

Set A₁: n=1, |A₁|=2, so i=1+2-1=2 → χ₂=1

Set A₂: n=2, |A₂|=3, so i=2+3-1=4 → χ₄=1

The binary encoding is: χ₀=1, χ₁=0, χ₂=1, χ₃=0, χ₄=1, χ₅=0, ...

Or written as a binary string: 10101...

Example 2.3 (Alternative View of Binary Encoding).

Consider a collection A = {A₀, A₁, A₂} where |A₀| = 1, |A₁| = 2, |A₂| = 3. Using the cumulative sum approach with proper accounting for prior set indices:

Positions where χᵢ = 1:

For A₀: i = 0 + |A₀| - 1 = 0 + 1 - 1 = 0

For A₁: i = 1 + |A₁| - 1 = 1 + 2 - 1 = 2

For A₂: i = 2 + |A₂| - 1 = 2 + 3 - 1 = 4

Therefore, χ₀ = 1, χ₂ = 1, χ₄ = 1, and all other positions are 0.

The resulting binary sequence is: 10101...

2.4 Binary Decoder and Recovery of Set Collections

Definition 2.3 (Binary Decoder).

The Binary Decoder D: {0,1}* → 𝒫(𝒫(ℕ)) is defined as:

For a binary string b = (b₀, b₁, b₂, ..., bₘ), D(b) = Å where:
Å = {Aₙ : bᵢ = 1 for i = n + |Aₙ| - 1, 1 ≤ n ≤ k}

where each Aₙ = {1, 2, ..., |Aₙ|} with |Aₙ| = i - n + 1 for each position i where bᵢ = 1.

Proposition 2.4 (Recovery of Set Cardinalities).

Given a binary sequence χ = (χ₀, χ₁, ..., χₙ₋₁), we can uniquely determine the cardinalities of the sets in the collection A = {A₀, A₁, ..., Aₘ₋₁} that generated it according to Definition 2.2.

Let I = {i | χᵢ = 1} be the set of indices where χᵢ = 1. Order these indices as i₀ < i₁ < ... < iₘ₋₁.

By Definition 2.2, we know that:

i₀ = |A₀| - 1

i₁ = |A₀| + |A₁| - 1

i₂ = |A₀| + |A₁| + |A₂| - 1

...

iₘ₋₁ = |A₀| + |A₁| + ... + |Aₘ₋₁| - 1

From these equations, we can solve for the cardinalities:

|A₀| = i₀ + 1

|A₁| = i₁ - i₀

|A₂| = i₂ - i₁

...

|Aₘ₋₁| = iₘ₋₁ - iₘ₋₂

Thus, we can recover the cardinalities of all sets in the collection.

Example 2.4 (Binary Decoding).

For the binary string b = 10101:

For each position i where bᵢ = 1:

i = 0: b₀ = 1 → Solve 0 = n + |Aₙ| - 1 → n = 1, |A₁| = 1 → A₁ = {1}

i = 2: b₂ = 1 → Solve 2 = n + |Aₙ| - 1 → n = 2, |A₂| = 2 → A₂ = {1,2}

i = 4: b₄ = 1 → Solve 4 = n + |Aₙ| - 1 → n = 3, |A₃| = 3 → A₃ = {1,2,3}

Therefore, D(10101) = {{1}, {1,2}, {1,2,3}}

Example 2.5 (Recovering Set Cardinalities).

Consider the binary sequence χ = (1, 0, 1, 0, 1). The indices where χᵢ = 1 are I = {0, 2, 4}.

Using the formulas derived in the proof of Proposition 2.4 and properly accounting for set indices:

For i = 0, with n = 0: |A₀| = i - n + 1 = 0 - 0 + 1 = 1

For i = 2, with n = 1: |A₁| = i - n + 1 = 2 - 1 + 1 = 2

For i = 4, with n = 2: |A₂| = i - n + 1 = 4 - 2 + 1 = 3

Therefore, the collection of sets A = {A₀, A₁, A₂} has the following cardinalities:

  • |A₀| = 1
  • |A₁| = 2
  • |A₂| = 3

Following the Binary Decoder convention from Definition 2.3, we can determine both the index of the set and its content:

For i = 0, χ₀ = 1:

  • Using i = n + |Aₙ| - 1, we get 0 = n + |Aₙ| - 1
  • This gives n = 0, |A₀| = 1
  • A₀ = {1}

For i = 2, χ₂ = 1:

  • Using i = n + |Aₙ| - 1, we get 2 = n + |Aₙ| - 1
  • This gives n = 1, |A₁| = 2
  • A₁ = {1, 2}

For i = 4, χ₄ = 1:

  • Using i = n + |Aₙ| - 1, we get 4 = n + |Aₙ| - 1
  • This gives n = 2, |A₂| = 3
  • A₂ = {1, 2, 3}

Therefore, we get the collection {{1}, {1,2}, {1,2,3}}.

Example 2.6 (Comparing Decoding Approaches).

Consider the binary string b = 10101 from Example 2.4. The indices where bᵢ = 1 are I = {0, 2, 4}.

Using the set cardinality recovery approach from Proposition 2.4:

|A₀| = 0 + 1 = 1

|A₁| = 2 - 0 = 2

|A₂| = 4 - 2 = 2

Following the Binary Decoder approach from Definition 2.3, we can determine both the index of the set and its content:

For i = 0, b₀ = 1:

  • Solve 0 = n + |Aₙ| - 1
  • This gives n = 1, |A₁| = 1
  • A₁ = {1}

For i = 2, b₂ = 1:

  • Solve 2 = n + |Aₙ| - 1
  • This gives n = 2, |A₂| = 1
  • A₂ = {1, 2}

For i = 4, b₄ = 1:

  • Solve 4 = n + |Aₙ| - 1
  • This gives n = 3, |A₃| = 2
  • A₃ = {1, 2, 3}

Therefore, when applying the Binary Decoder from Definition 2.3 to the binary string b = 10101, we get the collection D(10101) = {{1}, {1,2}, {1,2,3}}.

Note the difference in indexing between the two approaches:

  • In Proposition 2.4, sets are indexed starting with A₀
  • In Definition 2.3, sets are indexed starting with A₁

When properly accounting for this indexing difference, both approaches yield consistent results. The cumulative sizes of sets align with the positions of 1s in the binary string.

3. Constraints on Set Elements

Definition 3.1 (Index-Consistent Sets).

A collection of sets A = {A₀, A₁, ..., Aₘ₋₁} is said to be index-consistent if for any two sets Aᵢ and Aⱼ in the collection, and for any index k, if both sets have an element at index k, then they have the same element at that index.

Example 3.1.

For the binary sequence χ = (1, 0, 1, 0, 0, 1) with set cardinalities |A₀| = 1, |A₁| = 2, and |A₂| = 3, an index-consistent collection might be:

Let U = {e₀, e₁, e₂, ...} be a universe of elements. Then:

  • A₀ = {e₀}
  • A₁ = {e₀, e₁}
  • A₂ = {e₀, e₁, e₂}

In this case, the collection forms a nested structure: A₀ ⊆ A₁ ⊆ A₂

The key property is that the element at position 0 in all three sets is e₀, and the element at position 1 in both A₁ and A₂ is e₁.

4. Proposed Bijection Approach

4.1 Mapping Strategy

The proposed approach for establishing a bijection between integers and rational numbers consists of:

  1. Using the Binary Encoding Rule (χᵢ) to establish a mapping between set collections and integers (via binary representation)
  2. Using the Set Union Ratio (μ) to establish a mapping between the same set collections and rational numbers
  3. Combining these mappings to create a bijection between integers and rational numbers

4.2 Modified Union Ratio for Extended Range

Definition 4.1 (Inverted Set Union Ratio).

The Inverted Set Union Ratio μ⁻¹: 𝒫(𝒫(ℕ)) → [1,∞) is defined as:

μ⁻¹(Å) = ∑A∈Å |A| / |∪A∈Å A|

This inverted ratio allows representation of rational numbers ≥ 1.

5. Challenges and Limitations

5.1 Non-uniqueness Problem

The primary challenge to establishing a bijection is the non-uniqueness of the μ mapping. Different set collections can yield identical μ values, as demonstrated in the following theorem:

Theorem 5.1.

There exist distinct collections of sets Å ≠ ℬ such that μ(Å) = μ(ℬ).

Proof: Consider the following collections:

Å = {{1,2}, {1,2}, {1,2}, {1,2}}

ℬ = {{1,2}, {1}, {1}, {1}, {1}, {1}, {1}}

For collection Å:

|∪A∈Å A| = |{1,2}| = 2

∑A∈Å |A| = 2 + 2 + 2 + 2 = 8

Therefore, μ(Å) = 2/8 = 0.25

For collection ℬ:

|∪A∈ℬ A| = |{1,2}| = 2

∑A∈ℬ |A| = 2 + 1 + 1 + 1 + 1 + 1 + 1 = 8

Therefore, μ(ℬ) = 2/8 = 0.25

Thus, μ(Å) = μ(ℬ) despite Å ≠ ℬ.

5.2 Additional Requirements

For a valid bijection, the following conditions must be satisfied:

  1. Injectivity: Different rational numbers must map to different binary encodings
  2. Surjectivity: Every integer must correspond to some rational number
  3. Constructibility: A systematic method for constructing the set collection for any rational number
