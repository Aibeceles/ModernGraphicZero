State of ZAD:

ZAD is a graph structure of p's intended to represent an arbitrary power set.

Members of the power set are represented as the roots of each p (indices).

ZAD is a graph of defferenced p's with a rule for extending the set to include other p's of interest.

In its singular composition, A monomial with scalar 1 and max degree is differenced and expanded.  
Each depths of the expanded graph contains a set of determined/underdetermined p roots.
A depth set can or not contain a complete set definied by nCr.  The reason is algebraic. 

ZAD completes the power set by including other p's in the graph structure.
These other p's are defined such that when differenced, respective constant terms are the same.
